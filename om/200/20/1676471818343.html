                        安装Kibana

                        李广妮
                        此页内容
                        • 下载镜像
                        • 修改配置
                          • 创建文件夹

                        # 下载镜像

                        执行命令，指定下载版本

                        docker pull kibana:7.17.9

                        # 修改配置

                        # 创建文件夹

                        mkdir -p /mydata/kibana/config

                        # 创建配置文件

                        执行如下命令，创建kibana.yml 并插入文件内容

                        注意正确填写elasticsearch的ip地址

                        sudo tee /mydata/kibana/config/kibana.yml <<-'EOF'
server.host: 0.0.0.0
server.shutdownTimeout: "5s"
elasticsearch.hosts: [ "http://192.168.37.129:9200" ]
monitoring.ui.container.elasticsearch.enabled: true
EOF

                        # 启动服务

                        -v /mydata/kibana/config/kibana.yml:/usr/share/kibana/config/kibana.yml

                        docker run --name kibana -p 5601:5601 -v /mydata/kibana/config/kibana.yml:/usr/share/kibana/config/kibana.yml -m 256m --restart=always --memory-swap=512m  -d kibana:7.17.9

                        # 验证安装

                        # 命令查看

                        docker ps

                        # 浏览器查看

                        在浏览器中访问 IP地址:5601， 打开Kibana控制台

                        首页展示Welcome To Kibana 字样，点击Explorer On My Own即可
