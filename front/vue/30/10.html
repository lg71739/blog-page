                  Vue CLI 现已处于维护模式!

                  现在官方推荐使用 create-vue 来创建基于 Vite 的新项目。

                  另外请参考 Vue 3 工具链指南open in new window 以了解最新的工具推荐。

                  # 简介

                  Vue CLI(Vue 脚手架) 是 Vue.js 开发的标准工具, 是 Vue 官方提供的标准化开发平台

                  官网地址: https://cli.vuejs.org/zhopen in new window

                  # 安装

                  # 安装Vue Cli

                  全局安装@vue/cli

                  npm install -g @vue/cli

                  # 创建项目

                  切换到你要创建项目的目录，然后使用命令创建项目：

                  vue create xxxx

                  # 启动项目

                  运行如下命令启动项目

                  npm run serve

                  如果下载缓慢请配置 npm 淘宝镜像：npm config set registry http://registry.npm.taobao.orgopen in new window

                  Vue 脚手架隐藏了所有 webpack 相关的配置，若想查看具体的 webpakc 配置，请执行：

                  vue inspect > output.js

                  # 脚手架目录结构

                  .文件目录
├── node_modules 
├── public
│   ├── favicon.ico: 页签图标
│   └── index.html: 主页面
├── src
│   ├── assets: 存放静态资源
│   │   └── logo.png
│   │── component: 存放组件
│   │   └── HelloWorld.vue
│   │── App.vue: 汇总所有组件
│   └── main.js: 入口文件
├── .gitignore: git版本管制忽略的配置
├── babel.config.js: babel的配置文件
├── package.json: 应用包配置文件 
├── README.md: 应用描述文件
└── package-lock.json: 包版本控制文件

                  # 修改默认配置

                  • vue.config.js 是一个可选的配置文件，如果项目的（和 package.json 同级的）根目录中存在这个文件，那么它会被 @vue/cli-service 自动加载
                  • 使用 vue.config.js 可以对脚手架进行个性化定制，详见配置参考 | Vue CLIopen in new window
                  module.exports = {
    pages: {
        index: {
            // 入口
            entry: 'src/index/main.js'
        }
    },
  // 关闭语法检查
  lineOnSave:false
}

                  总结:关于不同版本的函数：

                  1. 关于不同版本的函数：

                    1. vue.js 是完整版的 Vue，包含：核心功能+模板解析器
                    2. vue.runtime.xxx.js 是运行版的 Vue，只包含核心功能，没有模板解析器

                  2. 因为 vue.runtime.xxx.js 没有模板解析器，所以不能使用 template 配置项，需要使用 render函数接收到的createElement 函数去指定具体内容
